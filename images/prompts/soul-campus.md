# Промты: «душевный кампус» (принятый стиль, выбор Ренаты)

Это утверждённый стиль hero-картинок: ЖИВЫЕ эмоциональные студенты на красивых
американских кампусах, тёплая осень, золотой час, фотореализм. Отклонены: «mag_wide»,
«mag_real», «mag_soul», «soul» — оставлены `bak_soul` + `campus` (см. `../reference/soul_campus_*.png`).

## Что делает картинку «хорошей» (по survivors)
- Тёплый натуральный свет, золотой час; осенняя листва, коллегиальная архитектура (кирпич, готика).
- Живые искренние эмоции: группы смеются, прыгают, празднуют, идут вместе, общаются.
- Допустимы тёплые ИНТЕРЬЕРЫ: светлый воркшоп/студия со стикерами, уютная библиотека.
- Неглубокая глубина резкости, editorial-съёмка. БЕЗ текста/логотипов/цифр.
- Фирменный синий (#13445D/#1A5CA8) как лёгкий акцент, НЕ форсить в каждый кадр.

## STYLE_PREFIX (уже в `scripts/gen_images.py`)
«Реалистичная качественная фотография: живые, естественно выглядящие молодые люди (студенты)…»

## Сцены-промты (давшие хороший результат / для новых батчей)
1. Группа разнообразных студентов смеётся и идёт вместе по осеннему кампусному дворику, золотой час, готическая архитектура, падающая листва.
2. Студенты сидят на лужайке кампуса с ноутбуками, учатся, тёплый осенний свет, кирпичные корпуса на фоне.
3. Друзья радостно прыгают/празднуют на лужайке кампуса, осенние деревья, золотой час.
4. Двое студентов оживлённо разговаривают и смеются на скамейке, осенний кампус, мягкий свет.
5. Студенты работают вместе в светлой тёплой студии-воркшопе со стикерами на стене, творческая энергия, искренние эмоции.
6. Студент уверенно идёт через кампус с рюкзаком, осень, тёплый золотой свет, исторические здания.
7. Компания делает общий кадр / искренний момент смеха на ступенях кампуса, осень.
8. Студенты в уютном читальном зале/лаунже, тёплый свет, увлечённые и довольные.

## Уроки по стилю (фидбэк Ренаты, копить)
- ОТКЛОНЕНО: глянцевая «editorial-съёмка», постановка, идеальные улыбки, модели — выглядит
  ИСКУССТВЕННО. Также отклонены батчи mag_wide/mag_real/mag_soul/soul.
- ПРИНЯТО направление: ДОКУМЕНТАЛЬНЫЙ репортаж — незапозированные моменты (не смотрят в камеру),
  плёночная зернистость 35мм, естественная кожа и мелкие несовершенства, как настоящий
  студенческий снимок (STYLE_PREFIX в `scripts/gen_images.py` уже переписан под это).
- БОЛЬШЕ красивых кампусов, особенно в духе Лиги плюща (готика, плющ, грандиозные библиотеки).
- РАЗНООБРАЗИЕ ситуаций обязательно (не только «группа людей»): лекция, учёба в библиотеке с
  МНОГО людей и разными позами, прогулка по кампусу, семинар, кафе, лаборатория, общежитие,
  выпускной, сезоны (осень/зима/весна), время суток. Не повторяться.
- Цель базы: ~100 принятых картинок в `reference/`.

## ГОЧА с API (OpenRouter)
HTTP 403 `Key limit exceeded (total limit)` = на ключе исчерпан лимит трат (НЕ рейт-лимит).
Поднять/снять лимит ключа на https://openrouter.ai/settings/keys и проверить баланс
https://openrouter.ai/credits (пополнить). Большой батч (~96) за раз упирается в лимит —
лучше дробить и следить за остатком.

## Как сгенерировать ещё (батч)
```bash
cd "$HOME/Documents/Global Generation/gg-skill" && source .env
python3 scripts/gen_images.py --prompt "<сцена из списка>" --n 2 --aspect 16:9 \
  --name soul --out images/_drafts/soul2
```
Отсмотреть в `_drafts/soul2/`, лучшее перенести в `reference/`, сюда дописать сработавшую сцену.
