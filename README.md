# gg-skill

Скилл для сборки документов портала Global Generation: усиленные Word-гайды для
абитуриентов из России и СНГ — с проверенными фактами, постраничными сносками,
реальными скриншотами и сгенерированными в фирменном стиле картинками.

Это НЕ один большой .md файл. Это репозиторий со структурой: оркестратор ведёт
по этапам, а правила лежат по отдельным папкам. Так документ не «разъезжается»
и правила легко править по одной.

## Структура
```
gg-skill/
├── SKILL.md                 оркестратор: 6 этапов, отправляет к нужной папке
├── rules/general-rules.md   общие правила для ВСЕХ документов
├── template/                каркас, дизайн-спека, чек-лист, эталон/болванка
├── tables/tables-standard.md стандарт таблиц
├── images/                  стиль картинок, референсы, промты, генерация
├── factcheck/               критерии качества для Perplexity
├── scripts/                 фактчек, иллюстрации, сборка .docx, генерация картинок
├── content/content-plan.md  список 21 документа и статусы
└── output/                  собранные .docx (в git не коммитятся)
```

## Принцип работы (как просил Лев)
1. Один документ за раз, до конца. Не пачкой.
2. Claude не переписывает один файл бесконечно — правки идут в нужную папку правил.
3. Рената — оператор: ведёт Claude по этапам из SKILL.md.

## Запуск
В Claude Code из этой папки сказать: «Продолжаем gg-skill».
Claude откроет SKILL.md и пойдёт по этапам.

## Ключи API
Скопировать `.env.example` в `.env` и вписать ключи (`.env` в git не попадает):
- `PPLX_API_KEY` — Perplexity (фактчек).
- `OPENROUTER_API_KEY` — OpenRouter, модель `google/gemini-2.5-flash-image` (картинки).
Перед запуском скриптов: `source .env`.

## Требования
- `poppler` (`pdftotext`, `pdfimages`): `brew install poppler`
- Python 3 + пакеты: `python-docx`, `matplotlib`, `pypdf`
  `python3 -m venv venv && source venv/bin/activate && pip install python-docx matplotlib pypdf`

## История
Этот репозиторий заменяет более ранние локальные версии
(`~/gg-docs-skill`, `~/Documents/Global Generation/gg-document-builder`,
`~/Desktop/Documents_GG/SKILL.md` — там всё было в одном большом файле).
Они оставлены как есть для истории.
